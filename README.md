CARA MENGGUNAKAN WEBSITE INI -->

Absen
pilih kelas
pilih jenis kehadiran siswa "hadir", "sakit", "ijin"
jika siswa "ijin" maka kolom alasan akan terbuka dan alasannya bisa dimasukkan disitu
jika tidak pilih apa-apa maka siswa itu akan dianggap "alpha"

Menambahkan kelas
Klik tombol "Tambah kelas"
alert akan muncul pada kolom masukkan nama kelas seperti "XII TKJ 2", "10-A", "10-TKJ 1"
setelah itu alert lain akan muncul dan masukkan nama file dengan format "nama_file.json"  contoh"xiitkj2.json", "xii_tkj_2.json"
Kelas yang baru ditambah akan muncul mulai pada bagian paling bawah list

Hapus kelas
pilih kelas yang ingin dihapus
pencet tombol "hapus kelas"
lalu alert akan muncul dan pencet ok
 
Tambah murid
pilih kelas yang ingin ditambahkan murid
pencet "tambah siswa"
alert akan muncul
masukkan nama siswa
pencet ok atau enter

Hapus murid
pencet tombol "Hapus Murid" pada murid yang ingin dihapus

Download absensi
pilih kelas yang absensi nya akan di download
pencet "download absen"
alert akan muncul
pencet "OK" untuk download dalam format .csv, pencet "CANCEL" untuk download dalam format .xlsx
